# Dreamscape - FPS + Survival mini game combination


## Gameplay Video
https://vimeo.com/1192108058?share=copy&fl=sv&fe=ci

## Controls
FPS: 
- [WASD] movement 
- [Space] jump
- [left click] shoot
- [right click] (in air) dash
- [e] interact

Survival:
- [WASD] movement 
- [Space] jump
- [left click] shoot
- [e] interact
- [tab] inventory
- [c] crafting menu

## Features
- two gamemodes
- FPS shooter gamemode - has score, 3 levels, win/die screen, clear objective and linear level design
- Survival gamemode - basic minecraft-esque gameplay loop, get materials, make tools, build objective and escape to win
 
## Known Issues
- on Itch.io you need to full screen to see the entire game
- on Itch.io the cursor sometimes stays on screen, and to resolve it you must click outside the game window then back in
